Privacy Policy

Last Updated: March 2026

This Privacy Policy describes how the Discord Bot ("the Bot") collects, stores, and uses information.

1. Information Collected
The Bot may collect and store the following information:
- Discord User IDs
- Discord Server IDs
- Streak statistics
- Server timezone configuration

The Bot does NOT collect:
- Email addresses
- Passwords
- Private messages
- Personal information outside Discord IDs

2. Purpose of Data Collection
The stored data is used solely to provide bot functionality such as:
- Tracking streaks between users
- Maintaining global rankings
- Managing server-specific settings

3. Data Sharing
The Bot does not sell, trade, or share collected data with third parties.

4. Data Security
Reasonable measures are taken to protect stored data. However, no system can be guaranteed to be completely secure.

5. Data Retention
Data may be stored as long as it is required for the functionality of the Bot.

6. Data Removal
Server administrators or users may request data removal by contacting the bot developer.

7. Changes to This Policy
This Privacy Policy may be updated from time to time. Continued use of the Bot means you accept the updated policy.
